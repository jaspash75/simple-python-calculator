# simple-python-calculator
## About

This is my first Python project and my first GitHub repository.

The project is a menu-driven calculator developed in Python that performs basic arithmetic operations using functions. It demonstrates the fundamentals of Python programming, including loops, conditional statements, dictionaries, and user-defined functions.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Menu-driven interface
- Runs continuously until the user chooses to exit

---

## Concepts Used

- Python Functions
- While Loops
- Dictionaries
- Conditional Statements (`if`, `elif`, `else`)
- User Input
- Basic Problem Solving

## How to Run

### Clone the repository

```bash
git clone https://github.com/jaspash75/simple-python-calculator.git
```

### Open the project directory

```bash
cd simple-python-calculator
```

### Run the program

```bash
python calculator.py
```
